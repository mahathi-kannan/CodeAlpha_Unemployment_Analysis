# Unemployment Analysis with Python - CodeAlpha Task 2

This repository contains my submission for **Task 2: Unemployment Analysis with Python** during my Data Science Internship at CodeAlpha.

## Project Insights
- **Socio-Economic Shock Analysis**: Measured the severe spike in unemployment across Indian states during March, April, and May 2020 due to the COVID-19 pandemic lockdowns.
- **Segment Breakdown**: Evaluated variations in employment metrics across Rural and Urban landscapes.

## Dataset Features Analyzed
- `Region` (State-wise distribution)
- `Estimated Unemployment Rate (%)`
- `Estimated Employed`
- `Estimated Labour Participation Rate (%)`
- `Area` (Rural vs Urban)

## Tech Stack
- **Language**: Python
- **Environment**: Google Colab
- **Libraries**: Pandas, NumPy, Matplotlib, Seaborn
